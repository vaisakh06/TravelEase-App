# TravelEase 🧳📱

TravelEase is an Android application designed to simplify and enhance the travel planning experience. This app helps users manage trips, track expenses, and explore destinations easily from their smartphone.

---

## 📲 Features

- 🗺️ Browse and discover travel destinations
- 🧾 Plan trips and itineraries
- 💰 Track travel expenses
- 📍 View locations on map
- 🔔 Get reminders for upcoming trips
